SustainCity AI

An AI-powered, smart civic management and municipal tracking portal designed for urban sustainability, civic issue routing, and real-time public diagnostics. SustainCity AI leverages advanced generative intelligence to automate complaint classification, route infrastructure issues to respective local departments, and offer an interactive community knowledge assistant.

---

Features

* Smart Complaint Routing: Citizens can report local civic issues (e.g., waste accumulation, water leaks, broken streetlights) along with image uploads. The integrated Gemini AI architecture automatically assesses severity and routes the ticket to the correct department.
* Public Dashboard: Real-time metrics tracking total resolved, active, and urgent/high-priority complaints with localized status badges.
* Interactive Knowledge Assistant: A built-in AI chatbot trained on system instructions tailored for Indian civic standards, waste segregation rules (dry vs. wet waste), water conservation, and sustainability.
* Admin Control Panel: Dedicated workspace for city administrators to track, view details, update complaint statuses, and close resolved structural tickets.

---

Tech Stack

* Backend Framework: Flask (Python 3)
* Production Web Server: Gunicorn (WSGI compliance)
* Database: MySQL (relational tracking with structured schemas)
* AI Integration: Google GenAI Core (gemini-2.5-flash)
* Frontend Template System: Jinja2, HTML5, CSS3
